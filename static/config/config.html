What are endpoints?

Endpoint is just a fancy word for "API key". They are used in order to configure the abcd client.

var endpoint = 'ead7300a-93ad-41f7-916e-bcee6517251c'; abcd.endpoint = endpoint; var experiment = abcd.experiment('Checkout process') experiment.variant('control', function () { console.log('control'); }) .variant('blue button', function () { console.log('blue button'); }) .variant('yellow button', function () { console.log('yellow button'); }) .start(function () { experiment.end(function () { experiment.reset(); }); });
